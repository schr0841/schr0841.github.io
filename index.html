Selected Projects

U of M Bootcamp Final Project

This project aims to develop a Python predictive model for diabetes diagnosis using a dataset containing patient health metrics. It leverages machine learning techniques to predict with up to 97% accuracy the presence of diabetes based on various health indicators, including gender, age, hypertension, heart disease, smoking history, BMI, HbA1c levels, and blood glucose levels. My contributions included developing SVM and deep learning models, as well as advanced decision tree models.

Medical Image Classification with Convolutional Neural Networks

With colleague Christine J. We trained a convolutional neural network (CNN) model from scratch to classify CT chest scan images as either cancerous or normal, and then investigated the added benefit of using pre-trained CNN models on the same dataset by employing transfer learning. Furthermore, we investigated model ensembling in general and its potential benefits and drawbacks.
